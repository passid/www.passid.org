Password Generator

  1. www.passid.org is hosting by github，and all password will generate by client's browsers. We don't collects any data.
  2. www.passid.org is an open source project, you can fork it in github
  3. We used sha512 and base64，and the hash function is base64(sha512(username+websit+salt))
  4. We adds some special words in base64, such as -@#~,.[]()!%^*$&
